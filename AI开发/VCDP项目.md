# 一、背景

该项目用于进行车辆服务编排，可以创建工程、ECU、总线、报文、信号信息，支持数据导入导出
# 二、框架

后端服务位于项目根目录的`./serve`目录下

使用SpringBoot框架，数据库使用内嵌式数据库H2，采用本地文件的持久化存储模式

数据库需要有schedule.sql文件定义dll数据库建表语句

使用mabatis的方式调用数据库，每个持久化对象都要配套有mapper接口和mapper.xml来定语sql语句

前端工程在项目根目录的`./ui`目录下

使用 VUE.js, 使用后端SpringBoot的内嵌式Tomcat来启动

需要在项目根目录下创建一个build.sh脚本，该脚本可以将前端工程自动打包到后端服务的静态资源目录，同时可以将后端服务打包成jar

# 三、开发流程
## 1. 项目工程页面

进入项目首页后，展示项目列表

工程以卡片网格形式展现，需要展现的字段有：名称、描述

项目很多时要支持分页查询

卡片上要有编辑、删除功能

工程列表页面需要有新增、批量删除

### 数据定义

Project
1. id，不可配置，自动雪花算法生成，不用前端展示
2. name，名称，必填，字符串类型
3. desc，描述，选填，字符串类型

## 2. ECU配置页面

点击进入工程后，左侧菜单要有ECU配置，点击后进入ECU列表页面

ECU列表页面，需要有新增、批量删除的功能

ECU也是网格化卡片展示的，显示名称name、款型type，卡片上需要有查看、编辑、删除的功能

新增、查看、编辑时都是个大弹窗，大弹窗里面有多个tab页：
1. ECU基础配置：
	1. id，主键id，不可配置，不用前端展示，自动雪花算法生成
	2. projectId，不可配置，不用前端展示、自动关联工程项目，新增时自动关联
	3. name，名称，必填
	4. desc，描述，非必填
	5. mac，必填，mac格式输入和校验，冒号自动显示不用输入，存储的时候是去掉冒号的十六进制字符串
	6. ip，必填，ip格式输入和校验，存储的时候是带.的字符串
	7.  port ，必填，端口号
	8. index，必填，代表部件索引号
2. 转发表通信配置，EcuForwardInfo：
	1. pFlashMemoryStartAddress，可配置，必填，十六进制，代表s19编排起始地址，前端填写时不用填写0x，在最前面固定展示0x，给后端的时候固定拼接0x
	2. pFlashMemorySizeLimit，可配置，必填，十六进制，表示编排限制大小，前端填写时不用填写0x，在最前面固定展示0x，给后端的时候固定拼接0x
	3. ramMemoryStartAddress，可配置，必填，十六进制，代表s19编排ram起始地址，前端填写时不用填写0x，在最前面固定展示0x，给后端的时候固定拼接0x
	4. ramMemorySizeLimit，可配置，必填，十六进制，表示编排限制大小，前端填写时不用填写0x，在最前面固定展示0x，给后端的时候固定拼接0x
	5. id，主键，不可配置，不用前端展示，自动雪花算法生成
	6. ecuId，不可配置，不用前端展示，自动关联ecu
	7. projectId，不可配置，不用前端展示，自动关联工程
3. Can接口配置，CanInterface：
	1. id，不可配置，不用前端展示，自动雪花算法生成
	2. projectId，不可配置，不用前端展示，自动关联工程
	3. ecuId，不可配置，不用前端展示，自动关联ecu
	4. interfaceName，接口名称，必填
	5. channelId，接口id，从0开始的整数，必填
	6. interfaceType，必填，接口类型：0-MCU直连CAN，1-LSW下挂CAN，需要创建枚举，下拉选择
4. Lin接口配置，LinInterface：
	1. id，主键id，不可配置，不用前端展示，自动雪花算法生成
	2. peojectId，不可配置，不用前端展示，自关联工程
	3. ecuId，不可配置，不用前端展示，自动关联ecu
	4. interfaceName，接口名称，必填
	5. channelId，接口id，必填，从0开始的整数
5. Eth接口配置，EthInterface：
	1. id，主键id ，不可配置，不用前端展示，自动雪花算法生成
	2. peojectId，不可配置，不用前端展示，自动关联工程
	3. ecuId，不可配置，不用前端展示，自动关联ecu
	4. interfaceName，接口名称，必填
	5. type，必填，0-百兆口，1-千兆口，需要生成枚举，下拉选择

ECU基础配置，转发表通信配置，这两个标签页，是固定配置输入窗口，其他接口配置标签页中需要有新增接口的按钮，新增后生成一行配置需要用户输入

最终在这个大弹窗的右下角有保存、和取消按钮，保存时需要校验所有必填项目，并一次性创建所有配置并自动关联
